# 更换apt安装源

更换安装源对于提高安装速度有奇效。

apt源的配置文件位于`/etc/apt/sources.list`文件中。

``` bash
# 备份一份
sudo cp sources.list sources.list.bak
# 打开替换原内容
sudo vim sources.list
# 更新
sudo apt-get update
```
